<h1 align="center">🌍 Power BI: AQI Analysis Dashboard</h1>

<h2>📌 Project Overview</h2>
<p>
This project presents an interactive <b>Air Quality Index (AQI) Analysis Dashboard</b> created using 
Microsoft Power BI. The dashboard provides detailed insights into air pollution levels across different 
cities and regions by analyzing AQI data and pollutant concentrations. It helps users understand air quality trends, 
identify polluted areas, and monitor environmental conditions effectively.
</p>

<h2>🎯 Problem Statement</h2>
<p>
Air pollution has become a major environmental and health concern worldwide. Monitoring AQI data manually 
can be difficult due to large datasets and multiple pollutants involved. This dashboard provides a visual 
and interactive solution to analyze AQI trends, compare pollution levels, and identify areas with poor air quality 
for better environmental awareness and decision-making.
</p>

<h2>📂 Dataset Description</h2>

<ul>
<li>City / Region Name</li>
<li>Date-wise AQI Data</li>
<li>PM2.5 Levels</li>
<li>PM10 Levels</li>
<li>NO2 Concentration</li>
<li>SO2 Concentration</li>
<li>CO Levels</li>
<li>Ozone Levels</li>
<li>AQI Category</li>
</ul>

<h2>📊 Dashboard Features</h2>

<ul>
<li>Interactive AQI Dashboard</li>
<li>City-wise AQI Comparison</li>

</ul>
